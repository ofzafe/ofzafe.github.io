How to install Maven on Ubuntu

Download Maven

Download Maven at Download. Choose the binary version.

Extract Maven

tar xvfz apache-maven-X.X.X.bin.tar.gz

Move the folder to the recently created to /usr/local

sudo mv -R apache-maven-X.X.X /usr/local

Configure the PATH

export /usr/local/apache-maven-X.X.X/bin:$PATH

Enjoy Maven!
